# City Events Manager

A full-stack database application designed to manage urban events, ticket bookings, and user registrations. This project uses a Java-based backend with a MySQL database to handle data relationships and persistence.

### Co-author: [@SteliosTsir](https://github.com/SteliosTsir)

## Project Overview
* **Language:** Java
* **Database:** MySQL
* **Connectivity:** JDBC (Java Database Connectivity)

## Features & Database Design
* **Event Management:** CRUD operations for event scheduling, venue assignment, and category management.
* **User & Booking System:** Handles user accounts and bridges the gap between attendees and events through a transactional booking system.
* **Relational Integrity:** Implements primary/foreign key constraints to ensure data consistency across multiple tables (Users, Events, Bookings, Venues).
